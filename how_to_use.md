# 怎么使用RemindAI

- 纯对话模式仅能当作传统的LLM使用
- 对话模式下新建一个纯对话，会保持上一个工作目录，需要手动在Clip栏取消
- 纯对话模式下打开一个工作目录，可以切换到工作目录模式
- 工作目录下手动添加memory.json文件可以手动修改权限
- 新建工作目录可以与指定工作目录同名，这样在工作目录下面添加memory.json
- 在制作Skill的时候，提供openai格式的tool.json可以注入工具给Agent
- 可以在工作目录下自建.toolshell/skills文件夹进行当前工作目录技能注入
- 如果某历史记录对应的工作目录被删除了，即使切换到此历史，模型上下文也没有了
- 安装一个全局技能时，最好在卡片上编辑对该技能的描述来增加技能查找相关性
- 历史记录不会帮你切回原工作目录，而是保留到上一次的工作目录（如果是工作目录模式），这样做是为了防止工作目录被你主动删除导致意外
- 超长对话或者遇到调用一直失败，直接右上角的新建对话即可
- 有确定性的长任务开loop，否则关闭即可
